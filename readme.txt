By enabling the integration between the Docebo E-Learning platform and WordPress, this plug­in transforms any WP site into an E-Learning portal.

== Description ==

Imagine the **best CMS** working with the **most innovative LMS** - this plug­in brings your dreams to reality!

Let's integrate your WordPress website with the **Docebo E­Learning platform**, and transform your online portal in the next­generation E­Learning portal.

Thanks to this plug­in you will be able to:

* **Embed online courses** on your WordPress website and **assign training** by using the WordPress dashboard
* Access **courses** and **training reports**, plus control the whole **E-Learning project** directly from the WordPress dashboard
* **Display courses**, assigned to logged-in users by using the WP widget
* **Import** in Docebo your users-database
* Enable the **Single Sign On** between WordPress and Docebo

== Installation ==

This section describes how to install the plugin and get it working.

Requirements: This plugin requires the "curl" php module enabled [http://www.php.net/manual/en/curl.installation.php](http://www.php.net/manual/en/curl.installation.php)

1. Upload `docebo` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Configure the plugin trough the "General settings" page in the plugin, you can retrive the info requested here enabling the worpress plugin into you Docebo lms
4. Now you can configure the blocks and widget aspect inside the module and use them in your WordPress installation

== Frequently asked questions ==

= How can I get my Docebo E-Learning platform? =

You can activate your Docebo E­Learning platform free trial at [www.docebo.com](http://www.docebo.com). You have 14 days to try all
the Docebo’s features and test the integration with your WordPress website.

The Docebo E­Learning platform comes with a Courses Marketplace where you can buy online courses to
be delivered to your workforce. Some of them are free of charge, check them out!
